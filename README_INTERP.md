Как работать с интерпретатором?
- Загружаете файл. Создаете в той же директории файл hello.txt, в котором пишете текст программы на модельном языке. Запускаете интерпретатор из командной строки без доп. параметров

Что умеет?
- Исполнит код, написанный в hello.txt, а при невозможности выведет ошибку (лексическую/синтаксическую/семантическую). В начале работы выведет текст hello.txt в командной строке в подтверждение работы с нужным файлом.
